Floral - Responsive Flower Website
A beautiful, modern, and fully responsive flower shop website built with HTML and CSS. This project showcases an elegant e-commerce interface for a flower delivery business with smooth animations and a clean design.
Sections
1. Header

Sticky navigation bar
Logo with brand name
Navigation links (Home, About, Products, Review, Contact)
Icons for wishlist, cart, and user account


2. Home Section

Hero section with background image
Call-to-action button
Animated content entrance

3. About Section

Video showcase with overlay text
Description of business values
"Learn More" call-to-action

4. Features Section

Four key business benefits displayed with icons
Free delivery, returns policy, offers, and secure payments

5. Products Section

Product grid layout
Discount badges
Product images with hover effects
Quick action buttons (wishlist, add to cart, share)
Price display with original and discounted prices

Customization
Colors
Modify the CSS variables in the :root selector:
:root {
    --pink: #e84393;        /* Primary color */
    --light-pink: #ff69b4;  /* Accent color */
    --dark-pink: #d63384;   /* Dark accent */
    --black: #333;          /* Text color */
    --light-color: #666;    /* Secondary text */
}
